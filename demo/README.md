# @react/carousel Demo

A comprehensive demo showcasing the features of the @react/carousel package.

## Features Demonstrated

- **Basic Carousel**: Simple image carousel with responsive breakpoints
- **Card Carousel**: Carousel with card-style content
- **Accordion Carousel**: Expandable carousel with controlled state
- **Custom Styling**: Carousel with custom CSS classes and styling
- **Empty State**: Handling of empty data arrays
- **Status Controls**: Loading, error, and success states
- **Responsive Design**: Automatic responsive behavior across breakpoints

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3001](http://localhost:3001) in your browser

## Package Features

This demo uses the local @react/carousel package with the following features:

- **React 19** compatible
- **TypeScript** support with strict typing
- **Tailwind CSS v4** integration
- **Radix UI** primitives
- **Responsive breakpoints** (xs, sm, md, lg, xl)
- **Infinite scrolling** support
- **Touch and keyboard navigation**
- **Customizable styling** with CSS classes
- **Loading and error states**
- **Server-side pagination** support
- **Tree-shaking** optimized bundle
